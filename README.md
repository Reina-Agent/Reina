# Reina

Reina 桌面 AI Agent 的发行仓库 — 本仓库仅用于发布安装包，不再包含源码。

Release distribution for the Reina desktop AI agent. This repository hosts
installers only; the source code is no longer published here.

## 下载 / Downloads

前往 [Releases](https://github.com/Reina-Agent/Reina/releases/latest) 获取最新版本：

- **Windows (x64)**: NSIS 安装包（支持应用内自动更新）与 portable 单文件版
- **macOS (Apple Silicon)**: dmg / zip

Grab the latest build from the
[Releases page](https://github.com/Reina-Agent/Reina/releases/latest):
Windows x64 (NSIS installer with in-app auto-update, plus a portable build)
and macOS arm64 (dmg / zip).
